# 💰 Finance Dashboard UI

A modern and interactive **Finance Dashboard UI** built to help users track, analyze, and understand their financial activity.

This project demonstrates strong frontend development skills including **state management, data visualization, role-based UI, and responsive design**.

---

## 🚀 Live Demo

https://finance-dashboard-six-liard.vercel.app/

---

## 📌 Features

### 📊 Dashboard Overview

* Summary cards for:

  * Total Balance
  * Total Income
  * Total Expenses
* Time-based activity chart (income trend)
* Category-wise spending breakdown (Pie Chart)

---

### 💳 Transactions Management

* View all transactions with:

  * Name
  * Category
  * Date
  * Amount
* Features:

  * 🔍 Search transactions
  * 📅 Date filtering (7 days / 30 days / all time)
  * 🔄 Sorting (by date or amount)

---

### 👤 Role-Based UI (Frontend Simulation)

* **Viewer**

  * Can only view data
* **Admin**

  * Can see action buttons (Add / Edit / Delete)
* Role switching using dropdown

---

### 📈 Smart Insights

* Highest spending category
* Total income, expenses, and net balance
* Basic financial trend insights

---

### 🎨 UI/UX Highlights

* Clean and modern interface
* Fully responsive design
* Smooth animations using Framer Motion
* Interactive components

---

## 🛠️ Tech Stack

| Technology        | Description                    |
| ----------------- | ------------------------------ |
| React             | Component-based UI development |
| Tailwind CSS      | Utility-first styling          |
| Recharts          | Data visualization (charts)    |
| Framer Motion     | Animations and transitions     |
| JavaScript (ES6+) | Application logic              |

---

## 🧠 Key Concepts Used

* State Management using React Hooks (`useState`, `useEffect`)
* Derived state (filtered & computed data)
* Conditional Rendering (role-based UI)
* Component-based architecture
* Data transformation for charts
* Dynamic filtering and searching

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/Fernandes-Samuel01/Finance-Dashboard.git
```

2. Navigate to the project folder

```bash
cd Finance-Dashboard
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```
